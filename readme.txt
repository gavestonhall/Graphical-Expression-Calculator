Run this program via Eclipse's graphical user interface to view GUI.
- Simply import the project and run Main

Run this program via a command line to view the ASCII interface.
- Easiest way to do this is via Eclipse
- Ensure class path is finding gui.Main
- Run > Run Configurations > Show Command Line
- Copy the command that Eclipse uses to run the program
- Paste and run that command in your terminal (in the root directory of the project)

Feel free to compile the documentation yourself if you require the html pages.
